# Agency Website

[Demo]()

A modern, responsive business agency HTML template.

## Features
- Responsive design
- Hero slider
- Services, About, Features, Projects, and Newsletter sections
- Font Awesome icons
- Clean, semantic HTML & CSS

## How to Run
1. Clone or download this repository.
2. Open `index.html` in your browser.
